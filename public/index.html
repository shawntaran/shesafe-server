Android SensorApp
SensorApp → Server → This Dashboard

1
Open SensorService.kt and set SERVER_WS_URL to your deployed server URL, e.g. wss://your-app.onrender.com/android
2
Build and run SensorApp on your phone. Tap START and grant all permissions. The notification will say "Streaming → Dashboard".
3
The Android live badge above turns green and charts start updating in real time. Works over any network — no shared Wi-Fi needed.
