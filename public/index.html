📱 Android SensorApp Setup
1
Install the SensorApp APK on your Android phone (built from Android Studio)
2
Make sure SERVER_WS_URL in SensorService.kt is set to wss://shesafe-server-1iw9.onrender.com/android
3
Open the app, grant all permissions, then tap Start Monitoring
4
The Android live badge in the sidebar turns green — sensor charts start updating
